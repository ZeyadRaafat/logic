# logic_learning

A Flutter project for learning and experimenting with logic concepts.

## Getting Started

1. **Clone the repository:**
   ```
   git clone https://github.com/your-username/logic_learning.git
   ```
2. **Navigate to the project directory:**
   ```
   cd logic_learning
   ```
3. **Install dependencies:**
   ```
   flutter pub get
   ```
4. **Run the app:**
   ```
   flutter run
   ```

## Features

- Image selection screen
- Logic exercises and examples
